### （1）

已知下列反应的热化学方程式：

① `2H₂S(g) + 3O₂(g) = 2SO₂(g) + 2H₂O(g)` ΔH₁ = -1036 kJ·mol⁻¹

② `4H₂S(g) + 2SO₂(g) = 3S₂(g) + 4H₂O(g)` ΔH₂ = 94 kJ·mol⁻¹

## 参考答案

### （1）

**ΔH₄ = +170 kJ·mol⁻¹**

根据盖斯定律，反应④ = (① + ②)/3 - ③：
ΔH₄ = (ΔH₁ + ΔH₂)/3 - ΔH₃ = (-1036 + 94)/3 - (-484) = +170 kJ·mol⁻¹

---

### （2）

**优点：** 高温热分解能够获得H₂，H₂可作燃料，且副产物少

**缺点：** 能耗高（需要持续高温）

---

### （3）

**H₂S平衡转化率：** 50%（或0.5）

**平衡常数Kp：** 4.76 kPa

解析：设起始H₂S为1 mol，Ar为4 mol，转化率为α

```
2H₂S(g) ⇌ S₂(g) + 2H₂(g)
起始：1 mol     0      0
转化：α         0.5α    α
平衡：1-α       0.5α    α
```

平衡时H₂S与H₂的分压相等，即物质的量相等：1-α = α，解得α = 0.5，即50%。

总物质的量 = (1-α) + 0.5α + α + 4 = 5 + 0.5α = 5.25 mol

各组分分压：
p(H₂S) = (0.5/5.25) × 100 = 9.52 kPa
p(S₂) = (0.25/5.25) × 100 = 4.76 kPa
p(H₂) = (0.5/5.25) × 100 = 9.52 kPa

Kp = [p(S₂) · p²(H₂)] / p²(H₂S)
   = [4.76 × (9.52)²] / (9.52)²
   = 4.76 kPa

---

### （4）

① **n(H₂S)∶n(Ar)越小，H₂S平衡转化率越高（越大）**

**理由：** H₂S的热分解反应为气体分子数增大的反应，恒压条件下不断充入Ar，导致H₂S的分压降低，相当于减压，平衡正向移动，H₂S的平衡转化率升高。

② **n(H₂S)∶n(Ar)=1∶9对应图中曲线：d**

**H₂S分压的平均变化率：** 24.9 kPa·s⁻¹
