# 化学平衡 · N₂O₄ ⇌ 2NO₂ 经典题

- 该文件是讲解一道题而非某个概念，所以把题干与参考答案前置，完整讲解过程保存在最后

## 题干

在 373K 时，把 0.5mol N₂O₄ 气体通入体积为 5L 的真空密闭容器中，立即出现红棕色。反应进行到 2s 时，NO₂ 的浓度为 0.02mol/L。在 60s 时达平衡，容器内压强为开始时的 1.6 倍。

已知：N₂O₄ 无色，NO₂ 红棕色。

反应式：

```
N₂O₄(g) ⇌ 2NO₂(g)
```

## 问题

下列说法正确的是（多选）

A. 前 2s 内 v(N₂O₄) = 0.01 mol·L⁻¹·s⁻¹

B. 在 2s 时体系内的压强为开始时的 1.1 倍

C. 在平衡时体系内含 N₂O₄ 0.25mol

D. 平衡时，如果压缩容器体积，可提高 N₂O₄ 的转化率

E. 平衡后压缩容器，NO₂ 的浓度先增大后减小，最终比压缩前大

---

## 参考答案

### 关键数据计算

设平衡时 N₂O₄ 转化了 x mol：

```
N₂O₄(g) ⇌ 2NO₂(g)
起始：0.5       0
转化：x         2x
平衡：0.5-x     2x
```

平衡时总物质的量 = (0.5 - x) + 2x = 0.5 + x

恒温恒容下，压强比 = 物质的量比：

P_平衡 / P_起始 = (0.5 + x) / 0.5 = 1.

解得：x = 0.3

所以平衡时：
- n(N₂O₄) = 0.5 - 0.3 = 0.2 mol
- n(NO₂) = 0.3 × 2 = 0.6 mol
- 总物质的量 = 0.8 mol

### 选项分析

**A. ❌**

前 2s 内，NO₂ 浓度从 0 变为 0.02 mol/L
v(NO₂) = 0.02 / 2 = 0.01 mol·L⁻¹·s⁻¹
v(N₂O₄) = 1/2 × v(NO₂) = 0.005 mol·L⁻¹·s⁻¹ ≠ 0.01

**B. ✅**

2s 时 NO₂ 物质的量 = 0.02 × 5 = 0.1 mol
消耗 N₂O₄ = 0.05 mol
此时总物质的量 = (0.5 - 0.05) + 0.1 = 0.55 mol
P_2s / P_起始 = 0.55 / 0.5 = 1.1 ✅

**C. ❌**

平衡时 n(N₂O₄) = 0.2 mol ≠ 0.25 mol

**D. ❌**

压缩容器体积 → 压强增大 → 平衡向气体分子数减少的方向移动 → 逆向移动 → N₂O₄ 转化率降低

**E. ✅**

压缩瞬间：体积减小，所有气体浓度同等增大 → NO₂ 浓度瞬间增大

随后平衡逆向移动（向分子数减少的方向） → 部分 NO₂ 结合成 N₂O₄ → NO₂ 浓度从峰值回落

但相比压缩前：虽然 NO₂ 的**比例**降低了，但由于体积变小，最终 NO₂ 的**浓度**仍大于压缩前

**答案：B、E**

## 完整推演记录（讲解原文）

> 以下为基于"清晰解释复杂概念"技能进行的逐步推演教学对话，保留问答原文。

---

### 第 1 帧：先别管反应式

这道题所有难点的根，都在同一个事实里：

```
一拆二，就变多；二合一，就变少
```

具体来说：**1 个 N₂O₄ 拆开变成 2 个 NO₂**。就是这么简单的数量关系——1 变 2。

反过来说，2 个 NO₂ 也可以撞在一起重新变成 1 个 N₂O₄：2 变 1。

### 第 2 帧：低配场景——弹簧小人

想象真空容器里有两种小人：

- **双人连体小人**（N₂O₄）：两个小人背靠背粘在一起，无色透明
- **单人自由小人**（NO₂）：挣脱开了，单个活动，红棕色

这是两件事，同时在进行，不是在"先拆完再结合"：

```
一对连体小人 → 挣开 → 变成 2 个自由小人（1 变 2）
两个自由小人 → 撞到 → 变成 1 对连体小人（2 变 1）
```

关键句：**同一个容器里，对象越多，它们撞墙壁的总次数就越多，压强就越大。**

### 第 3 帧：只看一个局部——追踪一个小人

**Frame 1：** 一个连体小人在容器里待着。

```
[🫂]    ← 双人连体，无色
```

**Frame 2：** 它挣开了，变成两个自由小人。

```
[🧑] [🧑]   ← 两个自由的，红棕色
```

分子数从 1 变成了 2，翻了一倍。

> 停问：如果一开始容器里有 5 对连体小人，过一会儿有 3 对挣开了，容器里的总个体数变成了多少？
>
> **用户回答：2+3×2=8**
>
> 对。2 对没挣开（2 个🫂）+ 3 对挣开后的 6 个🧑 = 8 个对象。

### 第 4 帧：压缩容器

> 题干中 D 和 E 都跟"压缩容器体积"有关。

**Frame 1（压缩前）：**

```
| 🫂  🧑 🧑  🫂  🧑 |
|  🧑  🫂  🧑  🧑    |   ← 容器很大，距离远
```

**Frame 2（压缩瞬间）：**

```
|🫂🧑🧑🫂🧑🧑🫂🧑🧑|   ← 容器突然变小
```

所有对象的密度瞬间增加。NO₂ 浓度（单位体积里自由小人的数量）瞬间增大——这就是 E 选项"NO₂ 浓度先增大"的原因。

然后因为太挤，🧑 更容易撞上另一个 🧑。

> 停问：撞上了会怎样？
>
> **用户回答：合体了**
>
> 对！这就是压缩后的第二步：压缩瞬间所有对象挤在一起，🧑浓度↑（先增大）；随后因为太挤，🧑更容易撞上另一个🧑 → 合体成🫂 → 🧑减少（后减小）。

### 第 5 帧：E 选项——最终浓度比压缩前大还小？

> 停问：最终比压缩前大还是小？
>
> **用户回答：错了**（认为"最终比压缩前大"这句话不对）

用数字看为什么实际上是对的：

**压缩前（5L 容器）：**
```
┌─────────────────────────┐
│  🧑  🫂  🧑  🧑  🫂  │
│  🧑  🧑  🧑            │  ← 5L 空间
└─────────────────────────┘
```
🧑 数量 6，浓度 = 6/5 = **1.2 只/L**

**压缩瞬间（2.5L）：**
```
┌─────────────────┐
│ 🧑🫂🧑🧑🫂🧑🧑🧑│  ← 2.5L
└─────────────────┘
```
浓度 = 6/2.5 = **2.4 只/L**（翻倍）

**合体后**（部分🧑🧑→🫂，数量减少）：
```
┌──────────────────┐
│ 🧑🫂🧑🫂🧑🧑🫂 │  ← 多了🫂
└──────────────────┘
```
🧑 数量降到 4，浓度 = 4/2.5 = **1.6 只/L**

1.6 > 1.2，虽然合体后少了，但空间也缩了一半，最终浓度仍大于压缩前。

> **追问：你怎么能假设只有一对合成了，不能是都合成了呢？**
>
> 你说得对，我随便说"假设只有一对合体"没有道理。正确问法是：到底凭啥决定合体多少对？

### 第 6 帧：为什么是"恰好那么多"对合体？

两件事的"碰撞要求"不一样：

**分家（🫂 → 🧑 + 🧑）**
- 一个🫂自己就能干，不等谁
- 跟挤不挤没关系
- 速率只取决于：有多少个🫂站着

**合体（🧑 + 🧑 → 🫂）**
- 两个🧑必须撞到一起
- 跟挤不挤关系非常大
- 速率取决于：单位体积内同时碰到两个🧑的概率

压小容器后：

- 分家速率：🫂浓度变 2 倍 → **2 倍**（线性）
- 合体速率：找第一个🧑概率×2，找到第一个后在旁边找第二个的概率也×2 → **4 倍**（平方）

合体速率（4 倍）跑得比分家速率（2 倍）快得多，于是吃掉一部分🧑变成🫂，直到两个速率重新相等。

**"恰好多少对"就是那个让分家和合体重新跑回一样快的数字。** 对应专业名称叫**平衡常数 K**——由温度定死，压缩改变不了它。

> 停问：这个回答能接受吗？
>
> **用户回答：好像可以**

### 第 7 帧：D 选项——压缩容器，转化率提高还是降低？

转化率定义：一开始有 100 个🫂，最后有多少个挣开变成🧑。

压缩后挤了 → 🧑撞🧑更容易 → 合体加速 → 更多🧑回到了🫂状态。🫂比以前多了，说明挣开的总比例**下降**了。

D 说"可提高 N₂O₄ 的转化率"——正好相反。

勒夏特列原理的一句话版本：**系统会削弱你的改变，但不会完全抵消。**

- 你压缩（压强增大）
- 系统往分子数减少的方向走（🧑+🧑→🫂）来对抗你
- 分子数减少 = 自由小人变少 = 🫂回弹 = 转化率**降低**

**D ❌**

### 总结

```
1 变 2 → 分子数变多 → 推高压强
2 变 1 → 分子数变少 → 削弱压缩
```

全题所有答案——B 的压强比、C 的平衡组成、D 的转化率、E 的浓度曲线——**全部从这一个 1↔2 数差里长出来。**
